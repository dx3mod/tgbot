# Contributing Guide

## Development 

### Requirements

- **OCaml** >= 4.14
- **Dune** build system
- **OPAM** package manager


### Setup

Setup your development environment.

```sh
# Create a switch for project
$ opam switch create . --deps-only
# Or just install deps
$ opam install . --deps-only
```
